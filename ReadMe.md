Install pyhton if you don't have it : 
-----------------------------------

sudo apt-get install python

Then install Tkinter : 
---------------------

sudo apt-get install python-tk


Finally start the application with the command : 
-----------------------------------------------

python PyConverter.py


Enjoy !
